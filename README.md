# 🤡z-design 组件演示

react UI 组件库

- 基于 `react` `react-hooks` `typescript` `sass` `storybook` 构建

## 👉 快速安装

install with npm

```
npm i zdesign --save
```

### 📙 使用指南

usage

```tsx
// load style
import "zdesign/dist/index.css";

// load component
import { Button } from "zdesign";
```
